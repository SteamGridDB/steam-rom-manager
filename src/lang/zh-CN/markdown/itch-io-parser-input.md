## itch.io Windows-on-Linux 安装驱动器重定向
在 Linux 上，即使通过 Proton/Wine 运行，Windows 应用程序的位置也会记录为 Windows 路径。 如果设置，此字段将替换游戏路径的根目录。 例如，这将把 `C:\\Path\To\Game.exe` 更改为 `<field value>/Path/To/Game.exe`。

该字段仅对 Linux 系统有效。
